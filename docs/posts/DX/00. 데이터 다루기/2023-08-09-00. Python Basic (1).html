ex1) 홀수 짝수 구분

lst = list(range(1,10))
lst
[1, 2, 3, 4, 5, 6, 7, 8, 9]

sol

odd = [i for i in lst if i % 2 == 1]
evn = [i for i in lst if i % 2 == 0]
odd
[1, 3, 5, 7, 9]
evn
[2, 4, 6, 8]

ex2) 특정 숫자가 입력되었을 때 각 자리의 숫자를 구하기

sol

num  = int(input())

a = num // 100

b = (num - a*100) // 10

c = num-(a*100+b*10)

print("백의자리는 {}, 십의자리는 {}, 일의 자리는 {}".format(a,b,c))
254
백의자리는 2, 십의자리는 5, 일의 자리는 4

2. 자료형

대표적인 자료형

int float bool str complex
3 3.14 True “강철” 3+2j
5 3.141551 False “이강철” 2-2j

형태변환

float \(\to\) int

- 아래의 경우는 형태변환이 되었으나 정보의 손실이 발생한 것임

a = 3.4

_a = int(a)
type(a), type(_a)
(float, int)

\(\text{bool} \to \text{(float, int)}\,\text{and}\, \text{(float, int)} \to \text{bool}\)

- bool \(\to\) int, float

a = True
_a1 = float(a)
_a2 = int(a)
type(a), type(_a1), type(_a2)
(bool, float, int)

- int, float \(\to\) bool

a1 = 1
a2 = 1.0

_a1 = bool(a1)
_a2 = bool(a2)
_a1, _a2, type(_a1), type(_a2)
(True, True, bool, bool)

- str \(\to\) bool

bool("강철")
True
bool("")
False

암묵적 형변환

True +1
2
1*1.0
1.0
True + True
2

형태변환이 불가능한 경우

complex1 = 3+ 0j
complex1
(3+0j)
float(complex1) ## 에러 발생

문자열의 사칙연산(O)

"X"*2
'XX'

문자열의 사칙연산(x)

"X" * "Y"

"X" - "2"

"X" / "y"

- 즉 더하기를 제외한 나머지 사칙연산은 문자형 변수에 적용되지 않는다.

3. 문자열 다루기 고급

f-string(요즘 쓰는 방식)

ex1

n = "이강철"
a = 28
s = 100.213141
print(f"{a}살의 {n}의 점수는 {s:.2f}입니다. ")
28살의 이강철의 점수는 100.00입니다.
