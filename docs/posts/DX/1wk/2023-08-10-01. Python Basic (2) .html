rjust, ljust, center

- 각각, 오른쪽, 왼쪽, 중앙정렬을 하고 공백을 만든다

a = "apple"

print("[",a.ljust(7),"]",sep="")
print("[",a.center(7),"]",sep="")

[apple  ]
[ apple ]

- 근데 문장 단위로는 적용되지 않는다….

extend

- 두개의 리스트를 더할 때 extend함수를 사용한다. (append는 리스트 오브 리스트로 붙여준다는점에서 차이가 명확함)

- append처럼 +와의 차이가 같음.

a = [1,2]
b = [3,4]
a.extend(b)
print(f'''
      a.extend(b) = {a}
      a = {a}
      ''')

      a.extend(b) = [1, 2, 3, 4]
      a = [1, 2, 3, 4]
      
a = [1,2]
b = [3,4]
a+b
print(f'''
      a + b = {a}
      a = {a}
      ''')

      a + b = [1, 2]
      a = [1, 2]

True

- 이러한 경우를 방지하기 위해 copy함수를 사용
